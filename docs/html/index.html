get_messages(current_user_id, thread_id, first=50, after_id=-1, before_id=-1)[source]¶

Fetches paginated thread messages in descending order. If both after_id and before_id are passed then only after_id will be used.

Parameters:
  • current_user_id (int) – the user requesting the messages
  • thread_id (int) – the id of the thread the messages are being requested
  • first (int) – the number of messages to return upto, default 50
  • after_id (int) – if passed returns all messages with id greater than after_id
  • before_id (int) – if passed returns all messages with id less than before_id
Returns:

list of thread messages
